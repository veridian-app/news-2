🔑 Obtener Refresh Token de YouTube OAuth2

Instrucciones:
  1. Ingresa tu Client ID y Client Secret de Google Cloud
  2. Haz clic en "Autorizar" para abrir la ventana de autorización
  3. Inicia sesión y autoriza la aplicación
  4. Pega el código que recibas en el campo "Código de Autorización"
  5. Haz clic en "Obtener Refresh Token"
  6. Copia el refresh_token de la respuesta
